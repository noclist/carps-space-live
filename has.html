The CSS parent selector we've been waiting for.

Form Validation

The form field container changes style based on input state ‚Äî no JavaScript needed.

.field:has(input:focus) { border-color: #3b82f6; } .field:has(input:valid) { border-color: #10b981; } .field:has(input:invalid:not(:placeholder-shown)) { border-color: #ef4444; }

Card Variants

Cards automatically adjust their layout based on whether they contain an image.

With Image

This card has an image, so it uses a vertical layout with the image on top.

Without Image

This card has no image, so it gets extra padding and a subtle background pattern.

.card:has(img) { padding: 0; } .card:not(:has(img)) { background: linear-gradient(...); }
